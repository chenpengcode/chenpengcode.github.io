Java SE 7 及以后的版本，构造函数中可以省略泛型类型：ArrayList<String> files = new ArrayList<>(); 省略的类型可以从 变量的类型推断得出。

编译器可以很好地利用这个信息。当调用 get 时，编译器不需要进行强制转换，另外，编译器可以进行类型检查，避免插入错误类型的对象。使程序具有更好的可读性和安全性。
